# Patent Architecture and Mathematical Claims

This document defines the highly specific, proprietary mathematical methodologies and architectural constraints that form the core claims of the Agentic AI Dropout Prevention System. These detailed technical mechanisms are designed to move the intellectual property from a generic software application to a novel, non-obvious technical process.

## 1. Cognitive Load vs. Behavioral Engagement Indexing Engine

The system calculates a real-time **Cognitive Struggle Index ($CSI_t$)** by synthesizing asynchronous video playback micro-events against real-time quiz hesitation decay.

### Formula Definition:
The $CSI_t$ is calculated by integrating the frequency of video-rewind events normalized by the local difficulty weight, and dividing by an exponentially decayed function of the user's hesitation time before committing a quiz answer.

$$ CSI_t = \gamma \left( \sum_{k=1}^{n} \frac{R_k \cdot W_{diff}}{H_k} \right) + (1-\gamma) CSI_{t-1} $$

Where:
* $R_k$ = Frequency of 10-second rewind micro-events during critical conceptual timestamps.
* $W_{diff}$ = A dynamic difficulty weight assigned to the video segment based on global aggregate error rates.
* $H_k$ = The delta time (hesitation) between the moment a quiz UI element becomes fully visible and the first meaningful interaction event (keystroke or click).
* $\gamma$ = An exponential smoothing factor calibrating the decay of the cognitive load memory.

## 2. Self-Correcting Intervention Router (Multi-Agent RL Framework)

Instead of generically utilizing LLMs for interventions, the system utilizes a proprietary Actor-Critic multi-agent routing architecture governed by a specific **Proximal Reward Scalar**. This mechanism prevents "over-intervention" fatigue.

### Architecture constraint:
When an anomaly (high $CSI_t$) is detected, the Actor Agent (an LLM node) generates an intervention (e.g., a simplified explanation or syllabus modification). The Critic Agent subsequently maps the intervention against the student's *Hazard Shift* ($\Delta h(t)$) over the successive 48-hour epoch ($T_{eval}$).

### Reward Function Definition:
The Actor Agent updates its policy $\pi_{\theta}$ based on the proprietary reward $R_{intervene}$:

$$ R_{intervene} = \lambda_1 \left( \frac{S(t + T_{eval}) - \hat{S}(t + T_{eval})}{\hat{S}(t + T_{eval})} \right) - \lambda_2 \cdot P_{fatigue} $$

Where:
* $S(t+T_{eval})$ = True observed survival probability (retention state) at the end of the evaluation epoch.
* $\hat{S}(t+T_{eval})$ = The counterfactual predicted survival probability *had no intervention occurred*.
* $P_{fatigue}$ = An interference penalty factor scaling linearly with the number of interventions triggered in the trailing 7 days.
* $\lambda_1, \lambda_2$ = Hyperparameters balancing retention gains against intervention annoyance.

## 3. Sandboxed Counterfactual Risk Analysis Engine

Prior to intervention deployment, the system autonomously simulates divergent futures.

### Mechanism definition:
Upon triggering the intervention threshold, the system executes three parallel sandboxes generating counterfactual vectors for the feature space:
1. Strategy $\alpha$ (*Action = Do Nothing*)
2. Strategy $\beta$ (*Action = Inject Generated Simplification*)
3. Strategy $\zeta$ (*Action = Dynamic Syllabus Downgrade*)

For each counterfactual projection $X'_{strategy}$, temporal convolution networks predict the respective dropout hazard $h(t|X')$. The system autonomously routes to the strategy producing the minimal calculated cumulative hazard.

## 4. Autonomous Pacing Governor

Instead of broadly "adapting learning pace," this system dictates a closed-loop curriculum modification.

### Mechanism definition:
The Pacing Governor structurally intercepts API GET requests for the next LMS module $M_{i+1}$. If the user's exponential error rate over the preceding $m$ modules exceeds an overload threshold ($\tau_{load}$), the Governor overrides the API response, replacing $M_{i+1}$ with an autogenerated, logically synthesized sub-module $M'_{remedial}$ built specifically containing variations of failed concepts from $M_i$.
